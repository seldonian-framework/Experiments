experiments.experiment_utils.generate_resampled_datasets¶

generate_resampled_datasets(dataset, n_trials, save_dir)¶

Utility function for supervised learning to generate the resampled datasets to use in each trial. Resamples (with replacement) features, labels and sensitive attributes to create n_trials versions of these of the same shape as the inputs

Parameters:

  • dataset (pandas DataFrame) – The original dataset from which to resample

  • n_trials (int) – The number of trials, i.e. the number of resampled datasets to make

  • save_dir (str) – The parent directory in which to save the resampled datasets

  • file_format (str) – The format of the saved datasets, options are “csv” and “pkl”
