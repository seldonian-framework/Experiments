experiments.perf_eval_funcs.probabilistic_accuracy¶

probabilistic_accuracy(y_pred, y, **kwargs)¶

For binary classification only. 1 - error rate. Use when output of model y_pred is a probability

Parameters:

  • y_pred – Array of predicted probabilities of each label

  • y – Array of true labels, 1-dimensional
